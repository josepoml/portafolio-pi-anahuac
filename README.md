# Portafolio de Desarrollo Web - Anáhuac

Este repositorio contiene una colección de prácticas y proyectos desarrollados durante el curso de Programación en Internet en la Universidad Anáhuac. El portafolio muestra la evolución desde los fundamentos de HTML hasta la creación de aplicaciones interactivas con JavaScript y multimedia.

## 🚀 Contenido del Portafolio

El proyecto está organizado en diferentes secciones que representan hitos de aprendizaje:

### 1. Fundamentos y Estructura
*   **Práctica 2 - HTML:** Estructura semántica básica y elementos fundamentales de HTML5.
*   **Práctica 3 - Tablas y CSS:** Diseño estructurado mediante tablas, navegación por enlaces y personalización visual con CSS.

### 2. Interactividad con JavaScript
*   **Práctica 4 - JavaScript 1:** Introducción a la lógica de programación (variables, tipos de datos y estructuras de control). Incluye ejercicios de evaluación y cálculo de factoriales.
*   **Práctica 5 - JavaScript 2:** Manipulación del DOM y gestión de eventos para crear interfaces dinámicas.
*   **Temporizador:** Aplicación interactiva de cuenta regresiva con retroalimentación visual.

### 3. Multimedia y APIs
*   **Práctica 6:** Integración de elementos de Audio/Video nativos de HTML5, uso de IFrames de YouTube y Google Maps.

## 🛠️ Tecnologías Utilizadas
*   **HTML5:** Estructura y contenido semántico.
*   **CSS3:** Diseño responsivo y estilización (Grid, Flexbox, Variables CSS).
*   **JavaScript (Vanilla):** Lógica del lado del cliente y manipulación del DOM.
